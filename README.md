# Portfolio – Andrii Yarovyi

Moderne mehrseitige Portfolio-Website mit React und Vite.

## Seiten

- **Startseite** – Hero, Featured Skills, Featured Projects
- **Über Mich** – Vorstellung, Interessen, Ziele, Motivation
- **Lebenslauf** – Bildungsweg, Kenntnisse, Sprachen, Tech-Stack
- **Portfolio** – Projektkarten mit Demo- und GitHub-Links
- **Kontakt** – Kontaktdaten, Formular, LinkedIn & GitHub

## Entwicklung

```bash
npm install
npm run dev
```

Öffnen Sie [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Anpassungen

- **Profilfoto:** Bild in `public/profile.jpg` ablegen und in `ProfilePhoto.jsx` einbinden
- **Inhalte:** `src/data/content.js`
- **LinkedIn / GitHub:** URLs in `content.js` und auf der Kontaktseite anpassen
